# 手动测试验证流程

> 这份文档模拟一个**全新用户**从零开始的完整流程：从 Git 拉代码、安装环境、配置、启动服务、到逐项验证所有功能。
>
> 按顺序从上往下做就行。每一步都写了：做什么、怎么做、预期看到什么。
>
> 自动化测试（124 个）已覆盖核心代码逻辑，本文档补充的是自动化测试无法触达的真实运行验证。

---

## 第一部分：环境准备（全新机器从零开始）

### T1：确认 Python 版本

```bash
python3 --version
```

**预期**：显示 `Python 3.11.x` 或更高版本。

如果版本太低或没装：
- Ubuntu/Debian：`sudo apt update && sudo apt install -y python3 python3-venv`
- macOS：`brew install python@3.12`
- Windows：去 https://www.python.org/downloads/ 下载安装，**勾选 "Add Python to PATH"**

### T2：确认 Git 可用

```bash
git --version
```

**预期**：显示 git 版本号（任何版本都行）。

没装的话：
- Ubuntu/Debian：`sudo apt install -y git`
- macOS：`xcode-select --install`
- Windows：去 https://git-scm.com/ 下载安装

### T3：安装 uv（依赖管理工具）

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

装完后**关掉终端重新打开**（或者执行 `source $HOME/.local/bin/env`），然后验证：

```bash
uv --version
```

**预期**：显示 uv 版本号。

> Windows 用 PowerShell：`powershell -c "irm https://astral.sh/uv/install.ps1 | iex"`

### T4：确认 curl 可用

```bash
curl --version
```

**预期**：显示 curl 版本号。

没装的话：`sudo apt install -y curl`（Ubuntu）

## 第二部分：拉取代码与安装

### T5：从 Git 拉取代码

```bash
git clone https://gitee.com/xiaozai-van-liu/OcrYoloEngine.git
cd OcrYoloEngine
```

**预期**：
- 代码下载成功，进入 `OcrYoloEngine` 目录
- `ls` 能看到 `src/`、`docs/`、`configs/`、`tests/` 等目录

### T6：安装基础依赖

```bash
uv sync
```

**预期**：
- 自动创建虚拟环境（`.venv/` 目录）并安装依赖
- 最后没有报错
- 耗时约 30 秒 ~ 2 分钟（首次需要下载依赖包）

### T7：安装完整依赖（含文字识别和目标检测）

```bash
uv sync --extra dev --extra ocr --extra yolo
```

**预期**：
- 安装成功，无报错
- ⚠️ 这一步会下载 PyTorch、PaddlePaddle 等大包，可能需要 5~15 分钟，占几个 GB 磁盘空间

### T8：验证安装——`--help` 不报错

```bash
uv run ocr-yolo --help
```

**预期**：
- 显示命令行帮助信息（`serve` 和 `infer` 两个子命令）
- **不应该**触发 torch/paddle 等大库的加载（几乎秒出）

## 第十六部分：Python 代码调用示例

### T41：用 Python 脚本调用服务

```bash
uv run python3 -c "
import base64, requests

# 读图片
with open('tests/fixtures/golden_scene.png', 'rb') as f:
    b64 = base64.b64encode(f.read()).decode()

# 发请求
resp = requests.post(
    'http://localhost:8000/v1/match',
    json={
        'image': {'base64': b64},
        'methods': ['template'],
        'templates': ['demo_block'],
    },
)

# 验证
data = resp.json()
det = data['method_results']['template']['detections']
print(f'状态码: {resp.status_code}')
print(f'检测到 {len(det)} 个目标')
if det:
    cx, cy = det[0]['center']
    conf = det[0]['confidence']
    print(f'中心坐标: ({cx}, {cy})，把握程度: {conf:.2f}')
    assert abs(cx - 90) <= 3 and abs(cy - 70) <= 3, '坐标偏差过大！'
    print('坐标验证通过')
"
```

**预期**：
- 状态码 200
- 检测到 1 个目标
- 中心坐标接近 (90, 70)
- 打印"坐标验证通过"
