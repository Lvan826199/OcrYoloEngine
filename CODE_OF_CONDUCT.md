# 贡献者行为准则

本项目采用 [Contributor Covenant（贡献者契约）](https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/) 第 2.1 版作为行为准则。

## 我们的承诺

为了营造开放、友好的环境，我们承诺：无论年龄、体型、身体健全与否、种族、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、宗教或性取向如何，参与本项目和社区的每个人都不会受到骚扰。

## 我们的准则

有助于营造积极环境的行为包括：

- 对他人表达善意与同理心
- 尊重不同的观点与经历
- 给予并优雅地接受建设性反馈
- 为自己的错误承担责任并向受影响者致歉
- 着眼于社区整体利益

不可接受的行为包括：

- 使用性化的语言或图像，以及任何形式的性骚扰
- 嘲讽、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可公开他人的私人信息

## 执行

如遇辱骂、骚扰或其他不可接受的行为，可通过 **Lvan826199@163.com** 向项目维护者举报。所有投诉都会被审查与调查，并作出适当且必要的回应。维护者有义务为举报者保密。
